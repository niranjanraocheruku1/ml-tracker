# Tracker App

Tracker App is a Daily Task tracking web application deployed at [ml-tracker-chi.vercel.app](https://ml-tracker-chi.vercel.app/).

## Live Demo

- Deployed app: https://ml-tracker-chi.vercel.app/

## Overview

Tracker App is built to help organize and monitormy daily tasks or work in one place. The app is intended for tracking daily tasks or work information so you can review progress and keep track of your progress.
